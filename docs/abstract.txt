ABSTRACT

Project Title: SoilHealth Predictor: Machine Learning for Soil Fertility
               Classification

Institution:   SRM Institute of Science and Technology
Department:    B.Tech CSE Artificial Intelligence and Machine Learning — Section A
Subject:       Data Science Mini Project

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Soil fertility assessment is a foundational requirement for efficient
agricultural planning, yet conventional laboratory-based testing remains
prohibitively expensive and time-consuming for smallholder farmers across
rural India. This project presents SoilHealth Predictor, a machine
learning-based classification system designed to determine soil fertility
levels — categorised as High, Medium, Low, and Poor — using seven
measurable physicochemical soil parameters: Nitrogen (N), Phosphorus (P),
Potassium (K), soil pH, moisture content, ambient temperature, and
electrical conductivity (EC).

A synthetic dataset of 2,200 labelled soil samples was generated in
accordance with the Indian Council of Agricultural Research (ICAR) nutrient
classification standards, using class-conditional Gaussian distributions
with realistic noise to simulate field variability. Four supervised
classification algorithms were trained and evaluated: Random Forest,
Decision Tree, K-Nearest Neighbours (KNN), and Gaussian Naive Bayes.
Model performance was assessed using accuracy, weighted F1-score, and
5-fold stratified cross-validation.

Random Forest achieved the highest test accuracy of 94.1% with a weighted
F1-score of 0.938 and a cross-validation mean of 93.5% ± 0.7%, confirming
strong generalisation. Feature importance analysis identified soil pH
(23.1%) and Nitrogen content (19.8%) as the most significant predictors
of fertility class. The system also provides an interactive prediction
interface that returns fertility classification and site-specific
fertilization recommendations from raw sensor inputs.

This project demonstrates that ensemble machine learning methods can
reliably classify soil fertility at scale, offering a cost-effective
and accessible alternative to traditional soil testing for agricultural
decision support.
